# **App Name**: GTS Migration Buddy

## Core Features:

- User Profile & Registration: Allows nurses to register, create, and manage their profiles, including professional qualifications and contact information.
- German Readiness Assessment: An interactive, multi-format assessment tool to evaluate a nurse's aptitude and current baseline for learning German, providing immediate feedback.
- Germany Nurse Career Guide: Provides structured information on nursing requirements, registration processes, visa procedures, and living in Germany.
- Personalized Language Resources: Suggests tailored German language learning resources based on assessment results and user profile using an AI tool.
- Curated Job Listings: Displays relevant nursing job opportunities in Germany, with filters for specialty and location.
- Pathway Progress Tracker: Enables nurses to track their assessment results, application statuses, and progress through learning modules.

## Style Guidelines:

- Primary color: A deep, professional blue (#1A4C99) to convey trust and authority.
- Background color: A very light, desaturated blue-grey (#ECF0F4) for a clean and calming aesthetic.
- Accent color: A vibrant, analogous cyan (#33CAE8) for highlighting interactive elements and calls to action, providing a sense of clarity.
- Body and headline font: 'Inter' (sans-serif) for its clear, modern, and highly legible appearance suitable for extensive professional content.
- Clean, modern line icons should be used throughout, subtly referencing medical themes, career progression, and international connectivity.
- A structured and responsive layout focusing on logical information hierarchy, allowing for easy navigation through guides, assessments, and job listings on various devices.
- Subtle and functional animations should be implemented for feedback on assessments, form submissions, and seamless transitions between different sections, enhancing user experience without distraction.
